🔗 Connect Gmail — One-Time Setup

Once set up, clicking Send Reminder will create a Gmail draft directly with the Excel attached — no manual steps.

1
Create a Google Cloud Project

Go to console.cloud.google.com → New Project → name it e.g. "vCard Reminders" → Create.

2
Enable the Gmail API

In your project → APIs & Services → Library → search "Gmail API" → Enable.

3
Configure the OAuth Consent Screen

APIs & Services → OAuth consent screen → choose Internal (since you're on @masterclass.com — no IT approval needed for Internal). Fill in App name, support email → Save.

Add scope: https://www.googleapis.com/auth/gmail.compose (create drafts only — cannot read or send).

4
Create an OAuth 2.0 Client ID

APIs & Services → Credentials → Create Credentials → OAuth client ID → Application type: Web application.

Authorized JavaScript origins — add the URL where you open this file:

  • If using GitHub Pages: https://yourusername.github.io
  • If running locally: http://localhost:8000

Copy the Client ID (looks like xxxx.apps.googleusercontent.com).

5
Host the file (Gmail OAuth requires a real URL)
  • GitHub Pages (recommended, free): push this HTML to a repo, enable Pages → get a URL like https://yourname.github.io/vcard-tool/
  • Local server: run python -m http.server 8000 in the folder → open http://localhost:8000
  • Vercel / Netlify: drag-and-drop the file for a free hosted URL
⚠️ Note on "From" address: Drafts are created under your own account (c-cristina.sison@masterclass.com). To send from procurement@masterclass.com, add it as a "Send As" alias in your Gmail Settings → Accounts, or ask IT to set up delegation.
